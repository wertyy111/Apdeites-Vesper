## Vesper Launcher 1.0.12

- Исправлен цикл автообновления, когда после скачивания installer снова запускал старую копию лаунчера.
- NetSparkle больше не релончит предыдущее расположение exe после обновления.
- В silent update installer теперь получает явную текущую папку через `/DIR`, чтобы обновлялась именно установленная копия.
