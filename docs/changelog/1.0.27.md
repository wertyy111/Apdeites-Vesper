# Vesper Launcher 1.0.27

- Fixed multiplayer skins across launcher versions. Custom skins now resolve through one shared offline UUID flow, so Vesper players see each other correctly in multiplayer on legacy and modern versions, including 1.16.5 and 1.21.1.
- Fixed false friend-connect offers for LAN-only worlds. Remote players no longer get a broken connect button when the host only published a local network address.
- Reduced launcher bloat by cleaning obsolete temporary files, stale Java runtimes, and old natives folders that could accumulate after updates.
