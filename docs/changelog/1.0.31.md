## Vesper Launcher 1.0.31

- Fixed common `Minecraft closed immediately after launch. Code: 1` cases with better diagnostics and one automatic repair attempt.
- Improved Java runtime download reliability and fallback handling for users whose Java installation breaks or downloads incompletely.
- Fixed legacy 1.16.5 compatibility libraries so older Forge/OptiFine launches no longer miss required bridge jars.
- Reduced launcher-caused lag by using safer memory limits and saner default performance settings.
