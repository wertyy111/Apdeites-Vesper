# Vesper Launcher 1.0.28

- Fixed the startup crash caused by a corrupted WPF update window resource. The launcher now opens the startup update dialog from code, so broken `.resources` files no longer block launch.
- Simplified the Vesper account screen. Login now opens directly without the extra submenu, while registration stays one click away in the same panel.
- Refined launcher UI sections, including cleaner settings and skin screens, with the latest account layout included in this release.
