# Apdeites-Vesper

## Описание
Apdeites-Vesper — это проект, который позволяет легко управлять обновлениями лаунчеров, предоставляя заметки и обновления в структурированном формате.

## Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/wertyy111/Apdeites-Vesper.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd Apdeites-Vesper
   ```
3. Установите зависимости (если есть):
   ```bash
   npm install
   ```

## Использование
- Вы можете добавлять изменения в директорию `docs/changelog` для документирования обновлений.
- Загрузите свои заметки о версиях в соответствующий файл.
